Popover Menu
Use popover--right, popover--left, popover--below, popover--above, popover--up and popover--down to control the direction.
popover--up and popover--down can also be used to overwrite popover--below or popover--above.
Direction of the Popover Menu needs adjusitng to avoid being clipped by the parent node that hides overflow content.
The "menu" use case
User can mouse over the content of the Popover Menu and perform actions like clicking on a link.
Use px2 or py2 in conjunction with popover-margin to create a gap that will not cause the mouse to exit.

The "tooltip" use case
User cannot mouse onto the Popover Menu.
Use mx2 or my2 in conjunction with popover-margin to create a gap that will cause the mouse to exit.
To increase the sensitivity of mouseout, use mx3 or my3 instead.
Remove title attribute for UI elements that already have popover.
